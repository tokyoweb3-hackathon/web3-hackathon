# プロジェクト名

評価 DAO

## 概要

既存の履歴書、職務経歴書を補う評価絵お可視化できる DAO ツール。
過去・現在の所属コミュニティー内の評価を時系列で閲覧することができる。

## Build Setup

```bash
# スマートコントラクト側
$ npx hardhat node
$ npx hardhat run scripts/deploy.js

フロントエンド側
$ npme install
$ npm run dev

```
